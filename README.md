# Graph helper for Burning Wheel Character Burner
Graphs for the Burning Wheel Lifepaths for a better overview while burning a character.

## Used Libraries
This project uses the graphology JavaScript library [GitHub Graphology Releases](https://github.com/graphology/graphology/releases), [Graphology Docs](https://graphology.github.io/).

It directly uses the released .js-file to enable it in a GitHub page.
